## <br>**MIPS Instruction format**
   ### <br>**R - Format**
   | op | rs | rt | rd | shamt | func |      
   |----|----|----|----|-------|------| 
            
            * ALU            func$rd,$rs,$st 
   ### <br>**I - Format**
   | op | rs | rt | value or offset |         
   |----|----|----|-----------------|        
   
            * ALUi           alui $rt,$rs,value                                
            * Data Tranfer   lw $rt,offset($rs) 
                             sw $rt,offset($rs) 
            * Branch         beq $rs,$tr,offset 
   ### <br>**J - format**
   | op | absolute address |
   |----|------------------|
   
            * Jump           j address
